# Примеры

В исходной системе примеры - один из главных источников качества: агент сверяет с ними структуру, объем, ритм и интонацию.

В публичный экспорт реальные примеры текущего проекта не перенесены, потому что они могут содержать частный контекст, имена, бренды и опубликованные материалы.

Перед использованием системы добавьте сюда свои эталонные тексты:

```text
examples/
├── articles/
├── essays/
├── guides/
└── <ваш-формат>/
```

Лучше добавить 2-5 сильных примеров на формат, чем много случайных текстов.
