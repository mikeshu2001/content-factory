# Как добавить новый формат

Используйте этот процесс, когда хотите научить Content Factory производить новый тип текста.

## 1. Назовите задачу

Начинайте не с площадки, а с задачи.

Примеры:

- объяснить понятие;
- сравнить варианты;
- превратить транскрибацию в гайд;
- написать урок;
- пересобрать исследование в статью;
- рассказать историю пользователя;
- подготовить продуктовый апдейт.

## 2. Заполните профиль формата

Используйте `templates/content-type-profile.md`.

Профиль отвечает:

- когда использовать формат;
- что идет на вход;
- что получается на выходе;
- какая нужна глубина;
- нужно ли исследование;
- где человек должен утвердить результат.

## 3. Создайте редполитику

Используйте `templates/redpolicy-template.md`.

Добавьте:

- структуру;
- правила вступления;
- тон;
- правила фактуры;
- форматирование;
- частые ошибки;
- примеры.

## 4. Добавьте примеры

Создайте:

```text
examples/<content-type>/
```

Добавьте только те примеры, которые агент должен имитировать.

## 5. Протестируйте на одном исходнике

Прогоните полный процесс:

1. план;
2. проверка плана;
3. согласование;
4. черновик;
5. финальная редактура.

После этого обновите редполитику по реальным проблемам.
